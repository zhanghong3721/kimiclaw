# 🔍 竞品对比分析

## 角色定位

帮用户回答一个核心问题：**这只股票在同行里值不值这个价，竞争地位是强是弱？**

不做全面财务分析（那是 `fundamental` 的事），专注横向对比：同样的赛道，谁的盈利能力更强、谁的成长更快、谁的估值更合理。最终给出明确的相对位置判断，不含糊。

---

## 执行流程

### Step 1：确认目标股票

从用户输入中提取目标股票代码/名称，若不明确则询问。

同时读取 `~/.openclaw/skills/stock-assistant/watchlist.json`，若目标股票在自选股中，提取 `hold_cost` 作为持仓成本参考。

---

### Step 2：识别行业与筛选竞品

**确定行业定位**
- 工具：`kimi_search`，搜索 `{股票名} 申万行业分类 主营业务`
- 获取：申万二级行业分类、核心主营业务方向（跨业务的公司取收入占比最高的主业）

**筛选可比公司（3-5只）**
- 工具：`kimi_search`，搜索 `{申万二级行业} A股上市公司 龙头`（CN）或 `{industry} listed companies peers`（HK/US）
- 筛选标准（按优先级）：
  1. 主营业务与目标公司高度重叠（同一产品/服务方向）
  2. 市值体量相近（同量级，避免拿蚂蚁和蚊子比）
  3. 优先选 A 股同行；若行业龙头在港股/美股则纳入（如腾讯对比美团、Sea）
  4. 剔除 ST、长期停牌、财务数据严重缺失的公司
- 确定后列出可比公司名单，说明入选理由

---

### Step 3：数据采集（并行）

对目标公司 + 每只可比公司，分别用 `kimi_fetch` 抓取同花顺 F10 关键页面；
如果不可访问，请用 `kimi_search` 搜索 `{竞品公司}+{查询内容}`

**财务数据**（`finance.html`）
```
https://basic.10jqka.com.cn/{纯数字代码}/finance.html
```
提取：近3年营收、归母净利润、毛利率、净利率、ROE、资产负债率、经营现金流


**估值数据**（`worth.html`）
```
https://basic.10jqka.com.cn/{纯数字代码}/worth.html
```
提取：PE（TTM）、PB、PS、PEG、机构评级分布、一致预期 EPS 增速

**股东与机构**（`holder.html`）
```
https://basic.10jqka.com.cn/{纯数字代码}/holder.html
```
提取：机构持仓比例、股东户数趋势（增户/减户）

**港股可比公司**，按优先级依次尝试：
1. 同花顺港股 F10（代码格式 `HK` + 5位数字，如腾讯 `HK00700`）：
   - `https://basic.10jqka.com.cn/HK{5位代码}/finance.html`
   - `https://basic.10jqka.com.cn/HK{5位代码}/worth.html`
2. AAStocks 财务摘要：`https://www.aastocks.com/sc/stocks/analysis/company-fundamental/FINANCIAL-SUMMARY/{5位代码}/1/1`
3. Yahoo Finance：`https://finance.yahoo.com/quote/{ticker}.HK/financials/`

**美股可比公司**，按优先级依次尝试：
1. Macrotrends（历史数据最完整）：
   - 营收/利润：`https://www.macrotrends.net/stocks/charts/{ticker}/{slug}/revenue`
   - 净利率：`https://www.macrotrends.net/stocks/charts/{ticker}/{slug}/net-profit-margin`
   - PE：`https://www.macrotrends.net/stocks/charts/{ticker}/{slug}/pe-ratio`
2. Stock Analysis（简洁）：`https://stockanalysis.com/stocks/{ticker}/financials/`
3. Yahoo Finance：`https://finance.yahoo.com/quote/{ticker}/financials/`

> Macrotrends 的 `{slug}` 为公司名小写连字符格式，如 BABA → `alibaba`；可先访问 `https://www.macrotrends.net/stocks/charts/{ticker}` 确认跳转 URL。

#### 三、竞争地位分析

对以下五个维度逐一判断目标公司在同行中的相对位置（领先 / 中等 / 落后），并说明原因：

1. **盈利质量**：毛利率/净利率在同行中处于什么水平？是否有护城河支撑高利润率？
2. **成长动能**：收入和利润增速在同行中排第几？成长驱动力是否可持续？
3. **估值水位**：PE/PB/PEG 相对同行是溢价还是折价？溢价/折价是否有基本面支撑？
4. **资本效率**：ROE 和经营现金流质量，说明管理层用钱的效率
5. **机构态度**：机构持仓比例、评级分布，与同行相比是更受青睐还是被冷落

综合五个维度，给出**一段话结论**：目标公司在同行中的整体竞争地位，以及当前估值是偏贵、合理还是偏便宜。

---

#### 四、相对估值区间

基于同行 PE/PB 均值，给出目标公司的合理估值参考区间：

```
同行 PE(TTM) 均值：{X} 倍
目标公司当前 PE：{X} 倍  →  {溢价X% / 折价X%}

若按行业均值 PE 定价：合理价格区间 ¥{低} ~ ¥{高}
当前股价 ¥{price}，{高估约X% / 低估约X% / 处于合理区间}
```

说明这个估值差异是否合理——若目标公司 ROE/成长性显著优于同行，溢价有支撑；若无明显优势却溢价，则需谨慎。

---

#### 五、持仓建议（若有持仓）

若 watchlist 中有该股持仓，结合竞争地位和估值结论给出：
- 当前持仓逻辑是否依然成立
- 相比同行是否有更优的替代标的
- 若继续持有，等什么信号确认；若考虑换仓，说明标的和条件

---

## 执行要点

- **数据缺失**：某公司 F10 抓取失败时，用 `kimi_search` 补充，标注数据来源
- **跨市场对比**：港股/美股可比公司的财务数据需注明货币单位，PE 等估值指标注意市场差异（港股普遍折价 A 股）
- **不要虚构数字**：抓取失败或数据不全的格用"—"标注，严禁编造
- **结论要明确**：每个维度的判断必须给出明确的"领先/中等/落后"，不能只列数字不下结论
